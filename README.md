# WackyTric


Built with:

- [React.js](https://reactjs.org/docs/getting-started.html)

## Getting Started

1. Install dependencies

   ```npm
   install
   ```

2. Fire up the server and watch files

   ```bash
   yarn dev
   ```

## Deployment

1. Start app
   ```
   npm run start
   ```
2. Test

   ```
   npm test
   ```

3. Build
   ```
   npm run build
   ```
4. Deploy!

   ```
   firebase deploy
   ```
